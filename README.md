# Filament-storage
A simple cost effective and filament saving method to store all your wonderful exciting filaments. Using 2 or 3 3D printed parts and a 12mm dowel you can also make a beautiful cost effective filament storage rack

## Designing
The geometry is relatively simple it is a rectangular box with wide feet to keep it stable when holding multiple spools. The holder has 2 holes going through and through making it only a single part to print instead of various parts to print for a long rack.

The middle pieces should placed approximately every 0.6M or 60cm which is 2 feet in freedom units. It is possible to not use the middle brace but after testing in fusion with weights the wood tends to bend or break.
